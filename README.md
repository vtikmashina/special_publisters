# Pablisters Demo

Статический промо-сайт спецпроектов Паблистеров.

## Структура

- `index.html` — основная страница
- `styles.css` — стили
- `script.js` — интерактив и анимации
- `assets/` — шрифты и изображения
- `texts-for-copywriter.md` — все тексты страницы для редактуры

## Локальный просмотр

```bash
python3 -m http.server 8000
```

Открыть:

```text
http://127.0.0.1:8000/
```

## Деплой

Сайт статический. Подходит любой nginx/static hosting.
